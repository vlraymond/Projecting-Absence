Tools of Analysis

  • Aylien - a suite of natural language processing, information retrieval and machine learning tools for extracting meaning and insight from text
  • AlchemyLanguage - a suite of natural language processing APIs for text analysis including entity extraction, sentiment analysis, keyword extraction, concept tagging
  • CartoDB - a location intelligence and visualization engine
  • Textexture - a tool that visualizes any text as a network
  • Overview - a tool for analyzing large sets of documents using a sophisticated search engine, entity detection, and topic-based document clustering
  • VOSViewer - a tool for constructing and visualizing co-occurrence networks of important terms extracted from a body of text
